Conference notes, speaker research, and runnable examples from
[[https://babashka.org/conf/][Babashka Conf 2026]] in Amsterdam.

* Overview

Second edition of babashka-conf (first was Berlin 2023). Held at OBA
Oosterdok, Amsterdam, the day before Dutch Clojure Days 2026.

Topics covered: Babashka, SCI, GraalVM native-image, CLI tooling with
=lambdaisland/cli=, TUI development, pod protocol, and static site
generation.

* Usage

Run an example:

#+begin_src shell
cd examples/01-basic-cli
bb greet.clj --help
#+end_src

Run a lab:

#+begin_src shell
cd labs/01-sci-sandbox
bb sci_repl.clj
#+end_src

Start a REPL:

#+begin_src shell
lein repl
#+end_src

* Emacs Integration

Load =babashka-conf-2026.el= for project helpers:

#+begin_src emacs-lisp
(load-file "babashka-conf-2026.el")
#+end_src

Keybindings (bind the command map to a prefix of your choice):

| Key | Command                           |
|-----+-----------------------------------|
| =l= | Run a lab file                    |
| =e= | Run an example script             |
| =j= | CIDER jack-in to lab nREPL        |
| =n= | Open conference notes             |
